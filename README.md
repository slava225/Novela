# Neon Streets — GTA2-style 2.5D web game

Лёгкий прототип аркадной игры в духе GTA2, но с более выраженным **2.5D рендером** (изометрическая проекция, объёмные здания, глубина сцены).

## Что улучшено
- Переведено с простого top-down на **2.5D изометрический вид**.
- Улучшена графика: более глубокий 2.5D-город, объёмные здания, тротуары, фонари, тени, пульсирующие точки задач и частицы.
- Улучшено управление:
  - плавный контроль скорости/поворота,
  - `SPACE` как handbrake/drift,
  - `SHIFT` как короткий boost с кулдауном.
- Сохранены и улучшены игровые механики: миссии pickup/dropoff, wanted-уровень, полицейская погоня, улучшенный NPC-трафик и пешеходы с поведением по тротуарам.
- Добавлены коллизии с городом: машина больше не "режет" здания, есть выталкивание от препятствий и более читаемая физика столкновений.

## Локальный запуск
```bash
python -m http.server 8080
```
Откройте в браузере: `http://localhost:8080`

## Управление
- `WASD` / `Arrow keys` — движение и поворот
- `Space` — ручник/дрифт
- `Shift` — буст
- `R` — рестарт

## Деплой на GitHub Pages
В репозитории есть workflow: `.github/workflows/pages.yml`.

1. GitHub → **Settings** → **Pages**.
2. В **Build and deployment** выбрать **Source: GitHub Actions**.
3. Запушить изменения в ветку.
4. Дождаться деплоя и открыть URL из Pages.

## Файлы
- `index.html` — разметка + HUD.
- `styles.css` — визуальный стиль UI.
- `game.js` — игровой цикл, физика, рендер 2.5D, миссии и AI.


## Если сайт "не обновился"
- Проверьте, что в репозитории включён **Pages → Source: GitHub Actions**.
- Убедитесь, что последний run workflow `Deploy static site to GitHub Pages` завершился со статусом **Success**.
- Откройте страницу в режиме инкогнито или сделайте hard refresh (`Ctrl+F5`), потому что GitHub Pages иногда кэшируется.
- Для проверки версии добавлена метка в правом нижнем углу: `build 2026.02.08-2`.

- В `index.html` включён cache-busting (`styles.css?v=...`, `game.js?v=...`), поэтому после деплоя браузер тянет новую версию ассетов.
